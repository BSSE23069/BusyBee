# BusyBee 🐝

BusyBee is a full-stack web application built using the **MERN (MongoDB, Express.js, React, Node.js)** stack. It provides a platform for users to book rides and allows individuals to register as drivers to offer transportation services.

## 🚀 Features
- **Dual Role System:** Users can toggle between "Passenger" and "Driver" modes.
- **Ride Booking:** Real-time interface for requesting rides.
- **Driver Dashboard:** Manage incoming ride requests and track earnings.
- **Authentication:** Secure login and registration for all users.
